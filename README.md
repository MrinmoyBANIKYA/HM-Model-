# 🧠 HMM-Based Stock Regime Detection & Forecasting — Research Preview
 
> **Status: Active Research · Target Publication: Q3 2025**
> An ongoing research initiative exploring Hidden Markov Models as a superior probabilistic framework for financial time-series prediction in Indian equity markets.
 
---
 
## 🔍 Overview
 
This repository contains the foundational experimental work for a research paper currently in progress. The core hypothesis: **Hidden Markov Models (HMMs) are systematically underutilised in quantitative finance**, particularly in emerging markets like India, where regime-switching behaviour (bull/bear/sideways) is more abrupt and policy-driven than in Western markets.
 
Unlike deep learning models that treat stock prediction as a regression problem, HMMs model the *latent market state* — recognising that prices are not generated by a single static process, but by a sequence of hidden regimes. This framing is not just more theoretically sound; our initial experiments on NIFTY 50 data suggest it is also **more accurate**.
 
---
 
## 📊 Benchmark Results (NIFTY 50, 2018–2023)
 
The HMM was benchmarked against three industry-standard models across four error metrics:
 
| Metric | HMM vs LSTM | HMM vs ARIMA | HMM vs RNN |
|--------|-------------|--------------|------------|
| APE    | **12.2% better** | Comparable | **16.3% better** |
| AAE    | **12.2% better** | Comparable | **16.3% better** |
| ARPE   | **12.2% better** | Comparable | **16.3% better** |
| RMSE   | **15.1% better** | Comparable | **17.1% better** |
 
> HMM outperforms LSTM and RNN on all metrics. Its ARIMA comparison reveals an interesting parity — motivating deeper investigation into hybrid architectures (see Roadmap).

## 🧪 Current Scope (v0.1 — Experimental)
 
This is the first working prototype, establishing:
 
- **Gaussian HMM** trained on OHLCV features of NIFTY 50
- Comparison pipeline across **LSTM**, **ARIMA**, and **RNN**
- Evaluation using APE, AAE, ARPE, and RMSE
- Visualisation of predicted vs. actual price trajectories
- Data sourced via `yfinance` for reproducibility

## 🗺️ Research Roadmap (Planned Extensions)
 
The next phases of research — to be integrated before journal submission — include:
 
### Phase 2 · Regime Characterisation
- Automatic labelling of hidden states as *Bull*, *Bear*, or *Sideways* using posterior decoding (Viterbi algorithm)
- Transition probability heatmaps for market regime visualisation
### Phase 3 · Hybrid HMM-LSTM Architecture
- Feed HMM-decoded regime labels as context features into an LSTM layer
- Hypothesis: regime-aware LSTMs will significantly outperform standalone LSTMs
### Phase 4 · Multi-Asset Generalisation
- Extend experiments beyond NIFTY 50 to sector indices (Bank NIFTY, NIFTY IT, NIFTY Pharma)
- Assess whether regime structures are sector-invariant or market-specific
### Phase 5 · Real-Time Inference API
- Package the trained HMM as a REST API
- Enable live regime detection with streaming market data
### Phase 6 · Paper Submission
- Target: *Journal of Quantitative Finance* or *Expert Systems with Applications*
- Working title: *"Latent Market Regime Detection Using Hidden Markov Models: Evidence from Indian Equity Markets"*

## 💡 Why This Matters
 
Most retail quant tools in India are built on black-box neural networks or lagging indicators. This research argues for a probabilistically interpretable, regime-aware alternative — one that a CFO or risk manager can understand and trust. The eventual goal is to make HMM-based regime detection accessible as an open-source tool for Indian markets.
